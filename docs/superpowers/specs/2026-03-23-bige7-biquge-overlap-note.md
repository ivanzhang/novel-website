# Bige7 与 Biquge 数据重叠说明

日期：2026-03-23

## 结论

当前本地数据表明，`storage/json/biquge` 基本可以视为 `storage/json/bige7` 的真子集。

最关键的定量证据：

- `biquge` 书目文件数：`1426`
- `bige7` 书目文件数：`200243`
- 按 `bookId` 文件名比对，`biquge` 的 `1426` 本全部都能在 `bige7` 中找到
- `bookId` 交集覆盖率：`100%`

这说明前一批 `biquge` 数据并不是独立库，更像是同一站群数据中的一个较小暴露面。
