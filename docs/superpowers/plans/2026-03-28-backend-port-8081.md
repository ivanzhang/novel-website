# Backend Port 8081 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把项目的后端/API 端口统一从 `3000` 切到 `8081`，前端本地测试继续固定在 `8080`，并让容器化部署与文档口径保持一致。

**Architecture:** 保持现有双容器结构不变，`backend` 容器只调整内部监听端口为 `8081`，`nginx` 继续作为对外唯一入口并反代到 `backend:8081`。前端继续使用运行时配置判定本地测试环境，但本地 `8080` 页面将改为请求 `http://localhost:8081/api`，生产同源仍走 `/api`。

**Tech Stack:** Node.js, Express, nginx, Docker Compose, 原生前端 HTML/JS, Node test runner
